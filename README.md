# DocuTrust AI

DocuTrust AI is an AI-powered document analysis and question-answering web application built using Flask and Google Gemini AI.

## Features

- Upload PDF documents
- Extract and analyze document content
- Ask questions about uploaded PDFs
- AI-powered responses using Google Gemini AI
- Simple and user-friendly interface

## Technologies Used

- Python
- Flask
- Google Gemini AI
- HTML
- CSS
- JavaScript
- PyPDF

## Project Structure

```
DocuTrust_AI

│

```

## How to Run

1. Install dependencies

```bash
pip install -r requirements.txt
```

2. Run the application

```bash
python app.py
```

3. Open browser

```
http://127.0.0.1:5000
```
